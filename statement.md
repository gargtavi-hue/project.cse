Project Statement: Student Budget Tracker

Problem Statement
University students often struggle to keep track of their daily expenses (food, transport, stationery), leading to poor financial management. Existing apps are often too complex or filled with ads. There is a need for a simple, lightweight, and offline tool to track personal spending.

Scope of the Project
1. The project is a Desktop CLI (Command Line Interface) application. It focuses on:
2. Inputting financial data securely.
3. Storing data permanently in a local file.
4. Retrieving and displaying data for review.

Target Users
1. University Students
2. Beginners looking for a simple finance tool
3. Users who prefer offline tools over cloud apps

High-Level Features
1.User Menu: A loop-based navigation system.
2.Transaction Logging: Capturing detailed expense data.
3.File Handling: Reading and writing to expenses.json.
4.Summary Generation: Mathematical aggregation of costs.
